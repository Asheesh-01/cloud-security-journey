# Day 44 — Security Learning Platforms

**Volume:** 01 — Computer and Internet Foundations
**Topic:** 20 — Security Learning Platforms
**Tools:** OpenVPN to connect to lab environments

---

## TryHackMe

Guided rooms. Browser-based. Walks through topics step by step.
Useful: legal environment, reinforces specific concepts.
Roadmap limit: 4 rooms MAXIMUM. Supplement only. Not primary learning.
Recruiters cannot see TryHackMe progress. GitHub is what matters.
When to use: specific concept reinforcement, not daily learning.

## HackTheBox

Minimal guidance. Figure out the approach independently.
Develops genuine problem-solving skill.
Roadmap rule: Volume 6 onward only.
Starting too early = copying walkthroughs without understanding.
Start with Sherlocks (defensive/forensics) not offensive boxes.
Sherlocks: log analysis, malware analysis, forensics — matches Volume 6.

## OverTheWire Bandit

No guided steps. Just a problem and man pages.
Levels 0-15 only — reinforces Linux command-line fundamentals.
Access: overthewire.org/wargames/bandit. SSH in. No registration.
Productive exam-season revision activity.
Develops genuine command fluency unlike guided tutorials.

## OpenVPN

Required for TryHackMe and HackTheBox lab connectivity.
Installation: sudo apt install openvpn -y
Usage: sudo openvpn filename.ovpn (download .ovpn from platform)
Security: on VPN you share network with other users.
Keep your own Kali VM secure while connected.

## The Core Rule

Platforms are supplements. GitHub is proof.
TryHackMe dopamine ≠ recruiter-visible progress.
Every platform day with no commit is an invisible day.

Use platforms: surgically, for specific concept reinforcement.
Do not use platforms: as primary learning, as comfort zone,
as a substitute for building projects and committing to GitHub.

Recruiter sees: GitHub commits, projects, blog posts.
Recruiter does not see: TryHackMe badges, room completion counts.
