## SOC Analyst

Daily: monitor SIEM alerts, triage incidents, escalate confirmed threats.
Tier 1: alert triage. Tier 2: investigation. Tier 3: threat hunting.
Tools: Splunk, Elastic, Wazuh, CrowdStrike, SentinelOne.
Entry point: Indian MSSPs hire freshers at Tier 1.
Salary: 3-6 LPA fresher, 6-12 LPA with 2 years + certs.
Roadmap path: Volumes 5-6 build directly here.

## Security Engineer

Daily: build and maintain security tools, automate security tasks,
integrate security into CI/CD, manage vulnerability scanning programs.
Different from SOC: analyst responds to alerts, engineer builds the tools.
Skills: Python, Linux, cloud, CI/CD, security tool administration.
Roadmap path: Volumes 4, 8, 9 build directly here.

## Penetration Tester

Daily: authorized attacks on client systems, web/network/cloud assessments,
social engineering, red team exercises, detailed remediation reports.
Not an entry-level role — requires 2-3 years foundation.
Salary: 6-12 LPA entry consulting, 20-40 LPA experienced cloud testers.
Roadmap path: Volume 11 Option C.

## Incident Responder

Daily: contain and eradicate active breaches, collect forensic evidence,
determine scope, remove persistence, restore systems, write post-incident report.
Lifecycle: PICERL — Preparation, Identification, Containment,
Eradication, Recovery, Lessons Learned.
Skills: forensics, malware analysis, memory forensics, network forensics.
Roadmap path: Volume 6 foundation, SOC experience leads here.

## Cloud Security Engineer — Primary Target

Daily: IAM policy design, CloudTrail monitoring, compliance as code,
automated remediation, DevSecOps integration, cloud incident response.
Requires: security + cloud architecture depth simultaneously.
Salary: 10-18 LPA fresher (strong portfolio), 20-40 LPA 2-3 years.
Roadmap path: Volumes 7, 8, 9, 11 Option A.
